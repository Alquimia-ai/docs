## Getting Started with Alquimia Runtime

Learn how to integrate with Alquimia Runtime's AI assistant API and start building intelligent conversations.

### Your First API Call

<AccordionGroup>
  <Accordion icon="code" title="Make your first chat request">
    Start a conversation with a specific AI assistant using our chat endpoint. Here's a simple example using cURL:
    
    ```bash
    curl -X POST https://runtime.apps.alquimiaai.hostmydemo.online/infer/chat/alquimia-twyd \
      -H "Content-Type: application/json" \
      -d '{
        "query": "Hello, how can you help me today?",
        "session_id": "my-first-session",
        "attachments": []
      }'
    ```
  </Accordion>
  <Accordion icon="javascript" title="Using JavaScript">
    Here's how to make the same request using JavaScript:
    
    ```javascript
    const response = await fetch('https://runtime.apps.alquimiaai.hostmydemo.online/infer/chat/alquimia-twyd', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        query: 'Hello, how can you help me today?',
        session_id: 'my-first-session',
        attachments: []
      })
    });
    
    const data = await response.json();
    console.log(data);
    ```
  </Accordion>
</AccordionGroup>

### Get Assistant Configuration

<AccordionGroup>
  <Accordion icon="settings" title="Retrieve assistant configuration">
    You can also get detailed configuration information for a specific assistant:
    
    ```bash
    curl -X GET https://runtime.apps.alquimiaai.hostmydemo.online/chat/alquimia-twyd
    ```
    
    This returns comprehensive information about the assistant's personality, behavior rules, and AI model configuration.
  </Accordion>
  <Accordion icon="javascript" title="Using JavaScript for configuration">
    ```javascript
    const configResponse = await fetch('https://runtime.apps.alquimiaai.hostmydemo.online/chat/alquimia-twyd', {
      method: 'GET'
    });
    
    const config = await configResponse.json();
    console.log('Assistant config:', config.config.nickname);
    console.log('Personality:', config.config.dante.profile.custom_clauses.personality);
    ```
  </Accordion>
</AccordionGroup>

### Understanding the Response

<AccordionGroup>

<Accordion icon="message-bot" title="Chat Response Format">
  The chat API will return a JSON response with the assistant's reply:
  
  ```json
  {
    "response": "Hello! I'm here to help you with any questions or tasks you might have. What would you like to know?",
    "session_id": "my-first-session",
    "timestamp": "2024-01-15T10:30:00Z"
  }
  ```
</Accordion>
<Accordion icon="users" title="Session Management">
  Use the same `session_id` across multiple requests to maintain conversation context. The assistant will remember previous interactions within the same session.
</Accordion>
<Accordion icon="cog" title="Configuration Response">
  The configuration endpoint returns detailed information about the assistant's setup, including personality traits, behavior rules, and AI model settings. This is useful for understanding how the assistant is configured and what capabilities it has.
</Accordion>

</AccordionGroup>
