Start a conversation with a specific AI assistant in Alquimia Runtime.

## Additional Endpoints

### Get Assistant Configuration

You can also retrieve detailed configuration information for a specific assistant using:

```
GET /chat/{assistant_id}
```

This endpoint returns comprehensive information about the assistant's configuration, including:

- **Stream and Control IDs**: For managing conversation flow
- **Assistant Configuration**: Profile, personality, and behavior rules
- **Integration Settings**: Tools, agents, and expert configurations
- **Dante Configuration**: Advanced AI model settings and custom clauses

### Key Configuration Fields

- **`stream_id`**: Unique identifier for the conversation stream
- **`control_id`**: Control identifier for managing the conversation
- **`config.nickname`**: Display name of the assistant
- **`config.description`**: Description of the assistant's purpose
- **`config.tags`**: Categorization tags for the assistant
- **`config.dante.profile.custom_clauses`**: Dynamic personality and behavior configuration
- **`config.dante.config`**: AI model configuration and integrations
